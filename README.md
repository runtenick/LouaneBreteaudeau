# Louane Breteaudeau Portfolio

Astro foundation for Louane Breteaudeau's architecture student portfolio website.

## Tech Stack

- Astro
- TypeScript
- Tailwind CSS
- Static site output

## Local Development

Use Node.js `>=22.12.0`. Node 24 is recommended and pinned in `.nvmrc`
for version managers such as `nvm` and `fnm`.

Install dependencies:

```sh
npm ci
```

Start the local dev server:

```sh
npm run dev
```

Because the site is configured for a GitHub Pages project path, open:

```text
http://localhost:4321/LouaneBreteaudeau/
```

Build the static site:

```sh
npm run build
```

Run Astro checks:

```sh
npm run check
```

The site source lives in `src/`. Future local portfolio content can be added under `src/content/`.
